📋 Como obter as informações necessárias

🔑 Como obter seu token

  1. Abra o Discord no navegador
  2. Pressione F12 (Ferramentas do desenvolvedor)
  3. Vá para a aba "Network"
  4. Recarregue a página (F5)
  5. Procure por requisições com "authorization"
  6. Copie o valor do header authorization

🆔 Como obter ID do servidor

  1. Ative o "Modo Desenvolvedor" no Discord
  2. Configurações → Avançado → Modo Desenvolvedor
  3. Clique com botão direito no servidor
  4. Selecione "Copiar ID"
